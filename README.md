<h1>占い予約サイト</h1>
プレビュー：https://astro-eight-blue.vercel.app/
<h2>【制作ポイント】</h2>
<P>・Contextによるコンポーネント間のデータ受け渡し</P>
<P>・多くのサイトで使われている、ヘッダやハンバーガーメニューの固定</P>
<h2>【制作経緯】</h2>
<p>同僚の知り合いの占い師の方が自分のウェブサイトを作ってくれる人を探しているという話を聞き、<br>依頼を受けられずともフォーム制作やサイトレイアウトの勉強になると感じ、依頼内容を想像しながら制作。</p>
<h2>【反省点】</h2>
<P>・勉強しながら制作したこともあり、かなりコードが纏まっていないため見づらい</P>
<P>・フォームの内容をメールアドレスに送信しようとしたがうまく実装できなかった。</P>
